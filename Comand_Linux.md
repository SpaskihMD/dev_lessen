>> добавляет информацию к уже существующей

                                grep
grep - команда ищет все вхождения в файл, проще говоря поиск по файлам и папкам
например grep "google" 123.txt   (grep "слово" документ) ищем слово google в документе 123.txt
-i Игнорирует регистр (большие/маленькие буквы) grep -i "hello" file.txt
-w ищет только целые слова а не части  grep -w "cat" file.txt
-v показывает строки без указанного слова  grep -v "error" log.txt — все строки без "error"
-n показывает номер строки grep -n "TODO" code.py  покажет строки с TODO и их номера
-c Считает количество совпадений grep -c "apple" fruit.txt — сколько раз встречается "apple"
-r Ищет рекурсивно во всех папках grep -r "main" ./project/ — ищет во всех файлах папки project    (топовая команда) 
-l	Показывает только имена файлов с совпадениями	grep -l "error" *.log — какие лог-файлы содержат ошибки
-L	Показывает файлы без совпадений	grep -L "TODO" *.py — файлы без пометок TODO
-H	Всегда показывает имя файла	grep -H "config" *.ini
-h	Не показывает имя файла	grep -h "config" *.ini
-o	Показывает только найденное слово, не всю строку	grep -o "[0-9]\+" file.txt — только числа
-A NUM	Показывает NUM строк после совпадения	grep -A 2 "Exception" log.txt — ошибка + 2 строки после
-B NUM	Показывает NUM строк до совпадения	grep -B 3 "error" log.txt — 3 строки до ошибки
-C NUM	Показывает NUM строк до и после	grep -C 2 "warning" log.txt — 2 строки до и после
--color		Подсвечивает совпадения цветом	grep --color=auto "error" log.txt

                                touch
# Создать один файл
touch filename.txt

# Создать несколько файлов сразу
touch file1.txt file2.txt file3.txt

# Создать файл с пробелом в имени (кавычки обязательно)
touch "my cool file.txt"

# Если important.log не существует, touch не создаст его (и не выдаст ошибку)
touch -c important.log

# Установить дату файла на 1 января 2025 года, 15:30 (Год, Месяц, День, Час, Минуты)
touch -t 202501011530 old_document.txt

# Взять время одно файла и так же присвоеть его другому файлу
touch -r файл_у_которого взять время файл_которому_присвоить_такоеже_время
например touch -r text.txt new_file.txt
