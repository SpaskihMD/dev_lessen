whoami - команда чтобы посмотреть под каким мы пользователем
pwd - показывает в какой мы деректории (папке)
ls - просмотр файлов в текущем каталоге
cat имя файла - команда чтобы прочитать содержимое файла

cd .. подняться на один уровень выше в каталоге
cd  без парметрова переведеть в домашнюю директорию

clear - очистить терминал
rm - удалить файл
ls -l полная информация о файле, права на него, кто создал когда (всё)
      так же можно указать имя файла
      

head -n команда выведит первые 10 строк файла которого можно указать (n=число)
tail -n команда выведит первые 10 строк файла которого можно указать (n=число)

| флажок для ставиться между командами

ls -l|head -10 выведеть первые 10 строк файлов в папке директории

ip a или ip addr  просмотр своего ip-адреса

nano имя файла - открыть файл на редактирование

less имя файла - открыть файл на просмотр чтобы можно было смотреть без курсора (закрыть просмотр на букву Q )
/имя файла - чтобы найти в файле инфу по поиску чезер /
