# 🔐 Настройка sudo в Debian

## 📋 Пошаговая инструкция

### 1️⃣ Проверяем конфигурацию
```bash
nano /etc/sudoers
```
**Должна быть строка:**
```
%sudo	ALL=(ALL:ALL) ALL
```
> Это означает, что можно раздавать права sudo

---

### 2️⃣ Переключаемся на root
```bash
su -
```
Вводим пароль root

---

### 3️⃣ Добавляем пользователя в группу sudo
```bash
usermod -aG sudo имя_пользователя
```
**Пример:**
```bash
usermod -aG sudo mihail
```

---

### 4️⃣ Перезагружаем систему
```bash
reboot
```

---

## ✅ Готово!

После перезагрузки пользователь сможет использовать `sudo`.

---

## 📝 Коротко

| Шаг | Команда |
|-----|---------|
| 1 | `nano /etc/sudoers` — проверить строку `%sudo ALL=(ALL:ALL) ALL` |
| 2 | `su -` — стать root |
| 3 | `usermod -aG sudo username` — добавить в sudo |
| 4 | `reboot` — перезагрузка |
